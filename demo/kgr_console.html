⚖ KGR Sestava — izberi AI za vsako vlogo

Vloge (skeptik, optimist, arbiter) so fiksne — VEGA protokol. AI-ji so zamenljivi. API ključi se hranijo lokalno v brskalniku.

SKEPTIK —
Aktivno išče napake, tveganja in pomanjkljivosti. Privzeto zavrne — odobri samo z močnim dokazom.
OPTIMIST —
Išče razloge zakaj sprememba deluje. Privzeto odobri — zavrne samo ob jasnem tveganju.
ARBITER —
Sintetizira obe mnenji. Ne glasuje — presodi ali je konsenz dosežen ali je potreben R2/R3.

🔑 Kako dobim API ključ?

Google Gemini BREZPLAČNO

aistudio.google.com/apikey
  1. Odpri Google AI Studio
  2. Prijavi se z Google računom
  3. Klikni "Create API key"
  4. Kopiraj ključ (začne se z AIza...)

DeepSeek KREDIT

platform.deepseek.com/api_keys
  1. Registriraj se na platform.deepseek.com
  2. Dodaj kredit (min $2)
  3. API Keys → Create new key
  4. Kopiraj ključ (začne se z sk-...)

OpenAI (GPT-4o) KREDIT

platform.openai.com/api-keys
  1. Registriraj se na platform.openai.com
  2. Dodaj kredit (min $5)
  3. API keys → Create new secret key
  4. Kopiraj ključ (začne se z sk-...)

Anthropic Claude KREDIT

console.anthropic.com/settings/keys
  1. Registriraj se na console.anthropic.com
  2. Dodaj kredit (min $5)
  3. Settings → API Keys → Create Key
  4. Kopiraj ključ (začne se z sk-ant-...)

🔒 API ključi se hranijo izključno v localStorage tvojega brskalnika. Ne pošiljajo se nikamor razen direktno do AI providerja med review-om.
