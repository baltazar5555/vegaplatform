Split Detection & R2 RebuttalZaznava razhajanj in R2 ugovor Auto
  • Unanimous (all agree) → skip R2, final verdict
  • Split (disagree) → R2 rebuttal round with cross-concerns
  • Severity divergence (gap ≥ 2 levels) → R2 even if verdicts match
  • R2 resolved → skip arbiter. Still split → arbiter breaks tie
  • Soglasno (vsi se strinjajo) → preskoči R2, končni sklep
  • Razhajanje (nestrinjanje) → R2 ugovorni krog s prekrižanimi pomisleki
  • Odstopanje resnosti (razlika ≥ 2 stopnji) → R2 tudi če se sklepi ujemajo
  • R2 razrešen → preskoči arbitra. Še vedno razdeljeno → arbiter prelomi
